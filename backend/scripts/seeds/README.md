# Seeds

Scripts que insertan datos iniciales o de prueba en la base de datos.

## Scripts disponibles

- `seed.py` — Seeding general
- `seed_program.py` — Programas de formación
- `create_master_admin.py` — Crea el usuario administrador maestro
- `create_test_loan.py` — Crea préstamos de prueba

## Ejecución

Desde `backend/`:

```bash
python scripts/seeds/seed.py
python scripts/seeds/create_master_admin.py
```
