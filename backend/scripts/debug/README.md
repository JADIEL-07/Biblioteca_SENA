# Debug

Scripts de diagnóstico. Sirven para inspeccionar el estado del sistema cuando algo
no anda como debería.

## Scripts disponibles

- `verify_db.py` — Verifica el estado de la BD
- `debug_admin.py` — Debug del usuario administrador
- `audit_system.py` — Auditoría del sistema

## Ejecución

Desde `backend/`:

```bash
python scripts/debug/verify_db.py
python scripts/debug/debug_admin.py
```
